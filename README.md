## RU: О проекте и правила

⚠️ **Проект еще не закончен!** Это развивающийся open-source блокчейн.

### 🍴 Правила для форков (Forks)
Вы можете свободно делать форк этого репозитория. Однако, **вы обязаны изменить название проекта**, если ваш форк создается для отдельного использования и вы не планируете отправлять свои изменения обратно в основной репозиторий (контрибьютить).

### 🤝 Правила для контрибьюторов
Если вы хотите помочь проекту, пожалуйста, при создании **Pull Request** и отправке **Commits** обязательно подробно указывайте информацию о том, что именно было изменено, добавлено или исправлено.

### 📜 Лицензия
Этот проект распространяется под свободной лицензией **GNU Affero General Public License v3.0 (AGPL-3.0)**. 

Это означает, что вы можете свободно использовать, модифицировать и копировать данный блокчейн-сервер. Однако, если вы изменяете код проекта и запускаете его на сервере (в том числе для предоставления удаленного доступа пользователям через сеть), вы **обязаны открыть исходный код** всех внесенных вами изменений под этой же лицензией.

Полный текст лицензии находится в файле [LICENSE](./LICENSE).

## EN: About and Rules

⚠️ **This project is not finished!** It is an ongoing open-source blockchain development.

### 🍴 Rules for Forks
Forks of this repository are welcome. However, **you must change the project name** if you do not plan to contribute your changes back to this repository and intend to use the fork independently.

### 🤝 Contribution Rules
If you are contributing to the project, please ensure that your **Pull Requests** and **Commits** include clear and detailed information about what has been modified, added, or fixed.
